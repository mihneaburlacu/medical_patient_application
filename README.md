MED PLAT - Mobile Health Monitoring App

About the Project
Med Plat is a native Android application designed to help users actively monitor their daily physiological data and maintain a direct line of communication with their healthcare providers. The app provides an intuitive interface for patients to log daily vitals, visualize historical health trends through interactive charts, and quickly access emergency contacts or their dedicated caregiver.
It includes features tailored for remote patient monitoring, including the ability to link a Fitbit account for seamless health data integration.

Key Features
User Authentication: Secure Login and Registration, including the ability to link a Fitbit ID.
Dashboard & Navigation: A clean, grid-based main menu and navigation drawer for quick access to core services.
Health Data Entry: Manual logging of daily physiological values including Heart Rate Variability (HRV), Blood Pressure (SBP/DBP), Oxygen Saturation (SpO2), Temperature, and Glucose levels.
Data Visualization: Interactive line charts that allow users to track their historical health metrics against established baselines.
Caregiver Portal: A dedicated profile page for the user's assigned caregiver, featuring one-tap quick actions to Call, SMS, or Email them.
Emergency Services: An SOS/Contacts module for immediate assistance.
Medical Resources: Access to medical topics from Department of Health & Human services US
Medical AI Assitant: Access to a built-in "Medical Assitant" for health queries

Technology stack
Language: Kotlin
UI components: Material Design
Dependency injection: Hilt
Navigation: Jetpack Navigation Component
Database: Room
REST API: Retrofit
JSON Parsing: GSON
Charting: MPAndroidChart
